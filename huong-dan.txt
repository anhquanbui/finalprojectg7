PHẦN 1: KẾT NỐI GITHUB

Cài GIT: https://git-scm.com/downloads/win

Mở VS Code > Ctrl+Shift+P > Gõ: GitHub: Sign in > Git: Clone dán link git@github.com:anhquanbui/finalprojectg7.git > chọn thư mục lưu


PHẦN 2: cấp quyền
 
Bước 1: Tạo SSH key

Trong Visual Studio Code, nhấn CTRL + Shift + ` để bật terminal.

Copy đoạn này bỏ vào:

ssh-keygen -t rsa -b 4096 -C "email@gmail.com" (email github của mình)

Khi được hỏi:

Save file: Nhấn Enter để dùng mặc định (C:\Users\<you>\.ssh\id_rsa)
Passphrase: Nhấn Enter 2 lần để bỏ qua (hoặc đặt nếu bạn muốn thêm bảo mật)

BƯỚC 2: Thêm SSH key vào GitHub

Copy và pass

cat ~/.ssh/id_rsa.pub

Truy cập: https://github.com/settings/ssh/new

Title: Ghi tên máy tính của mình
Key: Dán key vừa copy
Bấm Add SSH key

BƯỚC 3: Kiểm tra kết nối
ssh -T git@github.com
gõ Yes

có dòng Hi username! You've successfully authenticated, but GitHub does not provide shell access.


Từ thư mục dự án, copy và paste vào terminal:
git remote set-url origin git@github.com:anhquanbui/finalprojectg7.git

Sau khi làm xong (update file, bỏ file vào thư mục,...) thì nhấn CTRL + Shift + ` và chạy lần lượt

git add .
git commit -m "Update project via SSH"
git push
